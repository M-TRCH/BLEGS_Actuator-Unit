# วิธีคอมไพล์ไฟล์ lgs.tex ออนไลน์

## วิธีที่ 1: ใช้ Overleaf (แนะนำ - ง่ายที่สุด)

1. ไปที่ https://www.overleaf.com
2. สร้างบัญชีหรือเข้าสู่ระบบ
3. คลิก **New Project** → **Upload Project**
4. อัปโหลดไฟล์ `lgs.tex` จากโฟลเดอร์นี้
5. ตั้งค่า Compiler:
   - คลิก **Menu** (มุมซ้ายบน)
   - เลือก **Compiler: XeLaTeX**
   - คลิก **Recompile**
6. ดาวน์โหลด PDF ที่ได้

**หมายเหตุ:** หากฟอนต์ TH Sarabun New ไม่มีใน Overleaf สามารถ:
- เปลี่ยนเป็น `Angsana New` หรือ
- อัปโหลดไฟล์ฟอนต์เองไปยัง Overleaf

---

## วิธีที่ 2: ใช้ LaTeXBase

1. ไปที่ https://latexbase.com
2. เปิดไฟล์ `lgs.tex` ด้วย Text Editor
3. คัดลอกเนื้อหาทั้งหมด
4. วางในช่องแก้ไขของ LaTeXBase
5. เลือก **Compiler: XeLaTeX**
6. คลิก **Compile**
7. ดาวน์โหลด PDF

---

## วิธีที่ 3: ใช้ Papeeria

1. ไปที่ https://papeeria.com
2. สร้างบัญชีหรือเข้าสู่ระบบ
3. สร้างโปรเจคใหม่
4. อัปโหลดไฟล์ `lgs.tex`
5. ตั้งค่า Compiler เป็น **XeLaTeX**
6. คอมไพล์และดาวน์โหลด PDF

---

## วิธีที่ 4: ใช้ LaTeX Online API

เปิด URL นี้ในเบราว์เซอร์:

```
https://latexonline.cc/compile?git=https://github.com/YOUR_REPO&target=docs/theory/lgs.tex&command=xelatex
```

**หมายเหตุ:** ต้องอัปโหลดไฟล์ไปยัง GitHub ก่อน

## ปัญหาที่อาจพบ

### ฟอนต์ไม่รองรับ
หากบริการไม่รองรับ TH Sarabun New สามารถแก้ไขบรรทัดที่ 15 ในไฟล์:

```latex
\setmainfont{TH Sarabun New}
```

เปลี่ยนเป็น:

```latex
\setmainfont{Angsana New}
```

หรือ

```latex
\setmainfont{Noto Sans Thai}
```
